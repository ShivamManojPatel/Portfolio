Personal Portfolio

-------------------------------------
Description
-------------------------------------
A modern, responsive personal portfolio built with HTML, CSS, JavaScript, and Bootstrap, showcasing my education, projects, skills, leadership roles, experience, and contact information. This website acts as my digital resume and professional identity.

-------------------------------------
Live Demo
-------------------------------------
https://shivammanojpatel.github.io/Portfolio/

-------------------------------------
Features
-------------------------------------
Fully responsive UI
  - Designed using Bootstrap 4, the layout adjusts beautifully across phones, tablets, and desktops.

Clean Visual identity
  - Minimalistic light theme with professional typography, icons, and smooth section transitions.

The About Me section includes
  - Academic background
  - Projects
  - Experience & Leadership

Other sections include
  - Awards & honors
  - Known language, hobbies & skills
  - contact form (Google app script)
  - Google Maps embed

-------------------------------------
Technology used
-------------------------------------
Frontend
  - HTML5
  - CSS3
  - Bootstrap
  - jQuery
  - Themify icons

Backend (Contact form)
  - Google Apps script

Integrations
  - Google Maps JavaScript API
  - Mailto & Tel links
  - Resume PDF download
